PHP介绍

PHP（英文名：Hypertext Preprocessor，中文名：“超文本预处理器”）是一种通用开源脚本语言。利于学习，使用广泛，主要适用于web开发领域。

获取PHP语言的帮助

php.net

PHP环境安装

WAMP（Windows）（推荐）
XAMPP（Windows/Mac）
Mamp（Mac）（推荐）
Lamp（Linux）

构建虚拟主机

作用：能够快速定位需要使用的文件，不需要每次都繁琐的打上路径。例：
www.123.com/code/123.php
可以使用虚拟主机创建域名：
123.video（建议使用不常见后缀，常见后缀：com、cn等）
配上文件路径：
E:\wamp\www\code\123.php
之后访问123.video就可以直接使用123.php了

编辑器选择

Visual Studio Code（推荐）
Sublime Text 3（推荐）
phpstorm（收费）
