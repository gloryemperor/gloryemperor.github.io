PHP简介

PHP是一种创建动态交互性站点的强有力的服务器端脚本语言
PHP脚本在服务器上运行

PHP语法

基础PHP语法

PHP脚本可以放置于文档中的任何位置
PHP脚本以<?php开头，以?>结尾

PHP文件通常包含HTML标签及一些PHP脚本代码

PHP中的注释

PHP大小写铭感

在PHP中，所有用户定义的函数、类和关键词（if、echo等）都对大小写不敏感

但PHP中，所有变量都对大小写敏感

PHP变量

变量是存储数据的容器

类似代数x=5，y=6，z=x+y
PHP的名称可以很短，也可以取更具描述性的名称

变量规则

  • 变量以$符号开头，其后是变量的名称
  • 变量名称必须以字母或下划线开头
  • 变量名称不能以数字开头
  • 变量名称只能包含字母数字字符和下划线
  • 变量名称对大小写敏感

创建PHP变量

PHP没有创建变量的命令，变量会在首次为其赋值时被创建

注意：如果为变量赋值的是字符或字符串，要使用引号

PHP变量作用域

在PHP中，可以在脚本的任意位置对变量进行声明
变量的作用域指的是变量能够被引用/使用的那部分脚本
PHP有三种不同的变量作用域：

  • local(局部)
  • global(全局)
  • static(静态) 在上例中，有两个变量$x和$y,以及一个函数myTest。$x是全局变量，因为它是在函数之外声明的，而$y是局部变量，因为它是在函数内声明的
    如果我们在myTest()函数内部输出两个变量的值，$y会输出在本地声明的值，但是无法输出$x的值，因为它在函数之外创建。
    如果在myTest()函数之外输出两个变量的值，那么会输出$x的值，不会输出$y的值，因为它是局部变量，并且在myTest()内部创建

注意：可以在不同的函数中创建名称相同的局部变量，因为局部变量只能被在其中创建它的函数识别
